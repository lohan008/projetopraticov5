Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de demonstrar a criação de um site moderno utilizando HTML, CSS e JavaScript, integrado com Node.js e MySQL.

A aplicação conta com funcionalidades como:

  • Layout responsivo com CSS Grid
  • Menu lateral interativo
  • Sistema de login com banco de dados
  • Dark mode (modo escuro)
  • Integração com vídeos e imagens

Objetivo

Servir como base para projetos maiores, como sistemas web completos, dashboards administrativos e aplicações profissionais.

Tecnologias utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Node.js
  • MySQL

Próximos passos

Evoluir o sistema com autenticação segura, cadastro de usuários e integração com APIs externas.
